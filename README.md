## About

Single-page portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step — just fast, clean, and maintainable code.

**Highlights**
- Custom animated cursor and scroll-reveal system
- Typed role animation in the hero
- Animated skill bars driven by Intersection Observer
- Contact form powered by Formsubmit (no backend required)
- Fully responsive down to 320px

## Stack

`HTML5` `CSS3 (custom properties)` `Vanilla JS` `Google Fonts` `Formsubmit.co`

## Run locally

```bash
git clone https://github.com/francescjimenez/francescjimenez.github.io.git
cd francescjimenez.github.io
open index.html
```
